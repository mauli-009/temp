# Cars Dealership — Full-Stack Capstone Project

A national car retailer web application built with React, Django, Flask, Node.js, MongoDB, and SQLite. Deployed using Docker, Kubernetes, and IBM Cloud Code Engine.

## Tech Stack
- Frontend: React (JSX)
- Backend: Django + Django REST Framework
- Microservices: Flask (sentiment analysis), Node.js/Express (dealers)
- Databases: SQLite (cars/makes), MongoDB (dealers/reviews)
- DevOps: Docker, Kubernetes, IBM Cloud Code Engine, GitHub Actions
